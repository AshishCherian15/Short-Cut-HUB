Creator Story 👨‍💻

Ashish Cherian

Builder of ShortcutHub 🚀

Every morning started the same way: open laptop, hunt through tabs, search old bookmarks, lose momentum. 😵 I wanted a start page that feels like stepping into my own control room, not a cluttered shelf. So I started building ShortcutHub as a focused launchpad where speed, clarity, and personality can live together.

Idea to Solution 🧭

  • 💡 Idea: create one home for all essential daily websites.
  • 😵 Problem: default bookmarks get cluttered and break focus.
  • 🛠️ Approach: keep it editable, searchable, and visually calming.
  • 🚀 Solution: ShortcutHub with appbar actions, settings tabs, grouped shortcuts, and rich media backgrounds.
  • 🎯 Result: less friction, faster starts, and better daily flow.
